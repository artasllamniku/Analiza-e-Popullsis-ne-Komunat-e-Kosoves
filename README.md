# Analiza-e-Popullsis-ne-Komunat-e-Kosoves
Analiza e Popullsisë në Komunat e Kosovës Ky projekt realizon një analizë interaktive të popullsisë në komunat e Kosovës duke përdorur Python dhe Streamlit.
Për çka bëhet fjalë?
Ngarkim i të dhënave për komunat e Kosovës
Llogaritje e përqindjes së grave dhe të rinjve në secilën komunë
Vizualizim i të dhënave me grafikë dhe tabela
UI e lehtë për përdoruesin nëpërmjet Streamlit
Teknologjitë e përdorura
Python
Pandas
Streamlit
Matplotlib
